# Thumb Joystick Knob (SparkFun Thumb Joystick - Deluxe Compatible)

This repository contains 3D model data for a joystick knob compatible with the SparkFun Thumb Joystick - Deluxe.

## Included Files

- `thumb-joystick-knob.stl`: STL file for 3D printing.
- `thumb-joystick-knob.blend`: Blender source file.
